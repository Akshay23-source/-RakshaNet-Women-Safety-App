# 📱 Twilio Error 21265 - Quick Reference Card

## ❌ Error:
```
'To' number cannot be a Short Code: +9181205XXXX
```

## ✅ Cause:
**Twilio trial account** - Phone not verified

## ⚡ Quick Fix (5 min):
1. Open: https://console.twilio.com/us1/develop/phone-numbers/manage/verified
2. Click: "+ Add new Caller ID"
3. Enter: Your phone (+919876543210)
4. Verify: Answer call, enter code
5. Test: RakshaNet app
6. Done: SMS arrives! ✅
