## 💡 Key Concepts Explained

### What is Error 21265?
Twilio trial accounts can only send SMS to verified phone numbers. Not a bug!

### What is E.164 Format?
International phone format: `+[country code][number]`  
Example: `+919876543210`

### What is a Verified Caller ID?
A phone number you've proven you own by verification in Twilio Console.

### Trial vs Paid Account?
- **Trial**: Free, verified numbers only
- **Paid**: Pay-per-SMS, no restrictions
